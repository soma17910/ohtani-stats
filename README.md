# 日本人MLBスタッツトラッカー

MLB公式の [MLB Stats API](https://statsapi.mlb.com)(無料・キー不要)から、MLBの日本人選手の最新成績を取得し60秒ごとに自動更新するシングルページアプリです。上部のタブで選手を切り替えられます。サーバーもビルドも不要で、`index.html` 1ファイルだけで動きます。

## 対象選手

大谷翔平・鈴木誠也・岡本和真・吉田正尚・村上宗隆・山本由伸・佐々木朗希・千賀滉大・今永昇太・菊池雄星・菅野智之・松井裕樹・今井達也 ほか。選手は `index.html` 冒頭の `PLAYERS` 配列で追加・削除・並べ替えできます。URLハッシュ(例: `...#808967`)で特定選手を直接開けます。

選手の役割(二刀流/投手/野手)は自動判定し、表示するカードを出し分けます:
- **野手** — 打撃成績・打撃ログ・本塁打の累計チャート
- **投手** — 投手成績・登板ログ・奪三振の累計チャート
- **二刀流(大谷)** — 上記の両方

## 表示内容

- **直近の試合** — 試合中はLIVEバッジ+イニング・スコア、終了後はスコア、試合前は開始予定時刻
- **試合中の成績(リアルタイム)** — その試合の打撃ライン/投球ラインと打席ごとの結果。試合中は約25秒間隔で自動更新し、打席中はカウントも表示
- **打撃成績 / 投手成績**(今シーズン)— 主要スタッツに所属リーグ内の順位を併記
- **累計チャート**(本塁打 or 奪三振)— ホバーで試合ごとの詳細を表示
- **順位表** — 両リーグの3地区+ワイルドカード。所属チームをハイライト
- **直近の試合ログ**(打撃10試合 / 登板5試合)

シーズンは自動判定します(今年の成績が無いオフシーズン中は前年を表示)。ライト/ダークテーマはOS設定に追従します。

### 株価風の変動カラー

直近24時間以内に行われた試合による変動を試合ログから逆算し、上昇したスタッツは**緑▲**、下降したスタッツは**赤▼**で表示します。変動がない値、および直近24時間に試合がない場合は通常色です。端末に依存しない計算のため、どのデバイスで開いても同じ色になります。

## 起動方法

**方法1: そのまま開く**
`index.html` をダブルクリックしてブラウザで開くだけで動きます。

**方法2: ローカルサーバーで開く**
```powershell
powershell -ExecutionPolicy Bypass -File serve.ps1
```
→ http://localhost:8710/ を開く(ポートは `-Port 8080` などで変更可)

## カスタマイズ

`index.html` 冒頭のスクリプト内:

| 定数 | 意味 |
|---|---|
| `PLAYER_ID` | 選手ID(660271 = 大谷翔平)。他選手に変更可 |
| `REFRESH_MS` | 自動更新間隔(ミリ秒、既定 60000) |
